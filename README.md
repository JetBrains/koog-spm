# koog-spm

Swift Package Manager distribution for [Koog](https://github.com/JetBrains/koog) — the Kotlin AI agents framework.

This repository hosts the `Package.swift` manifest referencing XCFramework binaries
built and released by the main Koog repo. It is updated automatically on each Koog release.
